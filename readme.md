# Simple, a blog website project (back-end)

## Framework

1. Spring-boot 2.7.7 + Java 8
2. Spring Data JPA + Hibernate
3. Spring Security + JJWT

## Structure
```bash
simple
    ├───constant
    ├───controller
    ├───dto
    │   ├───request
    │   └───response
    ├───entity
    ├───repository
    ├───security
    │   ├───jwt
    │   └───services
    └───service
```
